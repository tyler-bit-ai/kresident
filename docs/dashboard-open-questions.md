# Dashboard Open Questions

구현 전에 사용자 확인이 필요한 항목이다.

## 1. 기타 국가군 기준

현재 구현 기본값은 `중국, 일본, 타이완, 미국, 홍콩, 싱가포르, 베트남, 필리핀, 말레이시아, 타이, 오스트레일리아, 캐나다, 인도네시아, 프랑스, 독일, 러시아(연방), 영국, 몽골` 외 모든 국가를 `기타`로 합산하는 방식이다.

추가 기준:

- 국가명은 원문 표시와 별개로 canonicalization 비교를 적용해 공백/구두점 표기 차이를 흡수한다.
- `기타`는 특정 대륙 그룹이 아니므로 `2019-12` 이후 원본에 `대륙` 컬럼이 있어도 `continentName`을 비워 둔다.

확인 필요:

- 이 기준을 그대로 유지할지
- 향후 국가군 목록을 직접 수정할 수 있게 할지

## 2. 단기 비자 비율 분모 해석

현재 구현 기본값은 `원본 엑셀의 국가별 총합계`를 분모로 사용한다.

확인 필요:

- 이 `총합계` 해석이 사용자 의도와 정확히 일치하는지
- 특정 시기 자료에서 예외 표기가 있을 경우 그대로 제외할지 보정할지

## 3. 상세 표 범위

현재 구현 기본값은 `국가/연도/월 멀티 선택 필터 결과 + 국가 검색어`를 모두 반영한 집계 표다.

확인 필요:

- 현재처럼 필터 결과 전체를 페이지네이션으로 보는 방식이면 충분한지
- 연 누계 또는 별도 요약 행이 추가로 필요한지

## 4. Excel 다운로드 범위

현재 구현 기본값은 `현재 화면 필터 결과 전체를 Excel로 다운로드`다.

확인 필요:

- 현재 방식으로 충분한지
- 차트별 다운로드 또는 전체 원본 dataset 다운로드가 추가로 필요한지
