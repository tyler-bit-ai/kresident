Guide

대시보드 도움말

이 대시보드는 무엇을 보여주나요?

법무부 출입국 통계월보의 외국인 입국자 원본 엑셀을 월별로 읽어, 단기 입국자와 국가별 입국 분포를 정적 대시보드로 정리한 화면입니다. 상단 필터를 바꾸면 아래 모든 차트와 상세 데이터가 같은 조건으로 함께 갱신됩니다.

입국 구분

전체는 B1, B2, C1, C3, C4를 합산한 단기 입국자 기준입니다. B1(사증면제)는 사증면제 입국만 따로 보고, B2(관광통과)는 관광통과 입국만 따로 봅니다. 단기관광객(B2제외)는 전체 단기 입국자에서 B2를 제외한 값입니다.

Filters

국가, 연도, 월을 함께 좁혀 원하는 구간만 볼 수 있습니다. 연도를 여러 개 고르면 월도 연도별로 따로 선택할 수 있어, 예를 들어 2025년 11·12월과 2026년 1·2월만 이어서 보는 식의 비교가 가능합니다.

Charts

시계열 차트는 선택 조건의 월별 입국자 흐름을 보여주고, 점에 마우스를 올리면 해당 월의 값이 나옵니다. 아래 차트들은 국가별 단기 비자 비율, 국가별 성별 비중, 국가별 단기 입국자 비중을 같은 필터 기준으로 비교합니다.

상세 데이터와 다운로드

상세 표는 현재 조건에 맞는 월별 국가 데이터를 내려서 보여줍니다. 우측 상단의 Excel 다운로드 버튼들은 현재 보고 있는 필터 결과를 그대로 파일로 저장합니다.
